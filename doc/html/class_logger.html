◆ isDebug()

bool Logger::isDebug ( )

Returns if debug log level is enabled.

This can be used in time critical situations to prevent unnecessary string concatenation (if the message won't be logged in the end).

Example:

if (Logger::isDebug()) {

Logger::debug("current time: %d", millis());

}
